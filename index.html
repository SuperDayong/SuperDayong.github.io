更换主题

1，首先在Git上寻找自己需要的主题，在博客目录 D:\hexo 下右键点击Git Bash，输入以下命令

git clone https://github.com/iissnan/hexo-theme-next themes/next

这样就完成了第一步，并且会出现 D:\HEXO\themes\next 这个文件夹

2，打开 D:\HEXO 目录下的 (下划线)config.yml 文件，将 theme 改为 next

保存并退出即可

3,本地测试

依次执行下面语句

hexo g #生成静态网页 
hexo s #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）

打开浏览器，输入： http://localhost:4000/

–这样本地的博客就建好了，怎么样，很简单吧！

4，下一步我们要做的就是部署

hexo deploy  #进行部署

然后可以使用Github用户名.github.io进行访问, 或者设置个性域名，最晚10分钟左右，即可看到自己的效果
